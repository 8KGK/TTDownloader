TikTok Downloader Bot
Telegram бот для скачивания видео с TikTok без водяных знаков.
Описание
Простой и удобный Telegram бот, который позволяет скачивать видео с TikTok. Просто отправьте боту ссылку на видео, и он автоматически скачает и отправит его вам.
Возможности

Скачивание видео с TikTok по ссылке
Автоматическая обработка и отправка видео
Удаление временных файлов после отправки
Простой и понятный интерфейс

Требования
Python
Telegram Bot Token (получить у @BotFather)
